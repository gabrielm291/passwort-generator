# Passwort Generator
Ein einfacher Passwort Generator, der dir im Browser ein sicheres und einmaliges Passwort erstellt. Gib die Länge an, Klein-/Großbuchstaben, Zahlen und Sonderzeichen auswählen und auf "Passwort generieren" klicken. Dein Passwort innerhalb von Millisekunden generiert und die Sicherheit wird dir anhand eines farblichen Rückmeldebalken gezeigt. Passwort kopieren und benutzen!
